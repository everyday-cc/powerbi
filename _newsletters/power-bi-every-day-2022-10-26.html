DAX functions of the day

{% raw %}SIGN (<number>){% endraw %} Determines the sign of a number, the result of a calculation, or a value in a column. The function returns 1 if the number is positive, 0 (zero) if the number is zero, or -1 if the number is negative.

↪ Microsoft

{% raw %}MAXX (<table>,<expression>){% endraw %} Evaluates an expression for each row of a table and returns the largest value.
