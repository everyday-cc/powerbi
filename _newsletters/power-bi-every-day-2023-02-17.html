{% raw %}Set up continuous data export from Azure IoT Central to a Power BI app{% endraw %} This module covers continuous data export from IoT Central. You use the scenario from your previously completed "Create your first Azure IoT Central app" Learn module, where you controlled a refrigerated truck. Here, in this module, you add continuous data export to both Azure Blob storage and an event hub. You then build a Power BI app to read and display the telemetry data on a mobile phone.

↪ Microsoft Learn

{% raw %}Clean, transform, and load data in Power BI{% endraw %} Power Query has an incredible amount of features that are dedicated to helping you clean and prepare your data for analysis. You will learn how to simplify a complicated model, change data types, rename objects, and pivot data. You will also learn how to profile columns so that you know which columns have the valuable data that you’re seeking for deeper analytics.

↪ Microsoft Learn
