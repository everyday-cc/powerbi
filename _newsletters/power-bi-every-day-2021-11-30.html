DAX functions of the day

{% raw %}ALL ( [<table> | <column>[, <column>[, <column>[,…]]]] ){% endraw %} Returns all the rows in a table, or all the values in a column, ignoring any filters that might have been applied. This function is useful for clearing filters and creating calculations on all the rows in a table.

↪ Microsoft

{% raw %}DOLLARDE (<fractional_dollar>, <fraction>){% endraw %} Converts a dollar price expressed as an integer part and a fraction part, such as 1.02, into a dollar price expressed as a decimal number. Fractional dollar numbers are sometimes used for security prices.
