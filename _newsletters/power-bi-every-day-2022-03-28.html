DAX functions of the day

{% raw %}DIVIDE (<numerator>, <denominator> [,<alternateresult>]){% endraw %} Performs division and returns alternate result or BLANK() on division by 0.

↪ Microsoft

{% raw %}AMORLINC (<cost>, <date_purchased>, <first_period>, <salvage>, <period>, <rate>[, <basis>]){% endraw %} Returns the depreciation for each accounting period. This function is provided for the French accounting system. If an asset is purchased in the middle of the accounting period, the prorated depreciation is taken into account.
